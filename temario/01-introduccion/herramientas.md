# Herramientas Online para Practicar Expresiones Regulares 📚

En la web podremos encontrar diferentes herramientas para practicar las expresiones regulares, a continuación se muestran algunos de ellos:

### Regex101
Una herramienta muy popular que proporciona un entorno de prueba en tiempo real para expresiones regulares. Ofrece una explicación detallada del patrón de regex y muestra las coincidencias en el texto de prueba. Soporta varios lenguajes de programación como PHP, JavaScript, Python y Go.

[visita el sitio web](https://regex101.com)


### RegExr
Esta herramienta permite escribir y probar expresiones regulares en un entorno interactivo. Incluye una biblioteca de expresiones regulares de muestra y una referencia rápida para metacaracteres, tokens, etc.

[visita el sitio web](https://regexr.com)


### Regex Tester - RegExp Pal
Es una herramienta simple y eficaz para probar y depurar expresiones regulares. Ofrece resaltado de sintaxis para hacer más visible cómo las expresiones regulares coinciden con el texto.

[visita el sitio web](https://www.regexpal.com)


### Rubular
Una herramienta basada en Ruby para probar expresiones regulares. Es útil especialmente para aquellos que trabajan con Ruby on Rails o lenguajes similares.

[visita el sitio web](https://rubular.com)


### RegexPlanet
Proporciona un entorno de prueba para varias plataformas de programación, incluyendo Java, .NET, Python y Ruby. Es útil para ver cómo una misma expresión regular se comporta en diferentes entornos de programación.

[visita el sitio web](https://regexplanet.com)


Estas herramientas son excelentes para practicar ya que proporcionan una forma interactiva y visual de entender y mejorar el uso de expresiones regulares. Además, muchas de estas plataformas ofrecen ejemplos y referencias que pueden ser muy útiles para aprender y recordar distintos aspectos de las expresiones regulares.
